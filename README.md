## English

Official release repository for **Wuji Glove HMI** - the desktop application for Wuji gloves.

### System Requirements

- **OS:** Ubuntu 22.04 or later
- **Architecture:** x64

### Download

Download the latest `.deb` package from the [Releases](../../releases) page.

### Installation

```bash
sudo dpkg -i wuji-glove-hmi_x.x.x_amd64.deb
```

If you encounter dependency issues:

```bash
sudo apt-get install -f
```

### Launch

Search for **Wuji Glove HMI** in your application menu, or run:

```bash
wuji-glove-hmi
```
